# 🚀 Internship JavaScript Projects Dashboard

A collection of front-end projects built using **HTML, CSS, and JavaScript** as part of my Internship Assignment.

## 📌 Overview

This project contains a centralized dashboard where all internship assignments can be accessed from a single place. Each project is developed using pure HTML, CSS, and JavaScript without any external frameworks or libraries.

The dashboard serves as the main entry point and provides easy navigation to all projects.
